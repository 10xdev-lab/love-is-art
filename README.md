# Love is Art - Kit de UI

Un kit de interfaz de usuario artístico construido con Tailwind CSS.

## Características

- Componentes modernos y elegantes
- Diseño responsivo
- Personalizable
- Optimizado para rendimiento

## Instalación

```bash
npm install
# o
yarn install
```

## Desarrollo

```bash
npm run dev
# o
yarn dev
```

## Construir para producción

```bash
npm run build
# o
yarn build
```
