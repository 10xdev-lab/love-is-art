# Love is Art - Kit de UI

Un kit de interfaz de usuario artístico construido con Tailwind CSS.

graph LR
    A[Inicio] --> B[Revisión]
    B --> C{¿Condición?}
    C -- Sí --> D[Acción 1]
    C -- No --> E[Acción 2]
    D --> F[Fin]
    E --> F[Fin]


## Características

- Componentes modernos y elegantes
- Diseño responsivo
- Personalizable
- Optimizado para rendimiento

## Instalación

```bash
npm install
# o
yarn install
```

## Desarrollo

```bash
npm run dev
# o
yarn dev
```

## Construir para producción

```bash
npm run build
# o
yarn build
```
